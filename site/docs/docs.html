Introduction

This API allows you to fetch random images from the server and display them in the user interface. We also provide code examples and recommendations for integrating it into your project.

Endpoints

1. Fetch a Specific Image

This endpoint retrieves a specific image based on a number provided in the request. 

GET /api/images/:any

Example request (for image number 123):


fetch("/api/images/123")
  .then((response) => response.blob())
  .then((blob) => {
    const imgURL = URL.createObjectURL(blob);
    document.getElementById("image-container").src = imgURL;
  })
  .catch((error) => console.error("Error fetching image:", error));

2. Fetch a Random Image

This endpoint retrieves a random image in Blob format.

GET /api/images-random

Example request:


fetch("/api/images-random")
  .then((response) => response.blob())
  .then((blob) => {
    const imgURL = URL.createObjectURL(blob);
    document.getElementById("image-container").src = imgURL;
  })
  .catch((error) => console.error("Error fetching image:", error));

3. Fetch Multiple Images

This endpoint allows you to fetch a specified number of random images. The "quantity" parameter defines the number of images you want to retrieve (1 to 10 images). 

GET /api/images-quantity/:quantity

Example request (for 5 images):


fetch("/api/images-quantity/5")
  .then((response) => response.json())
  .then((images) => {
    images.forEach((imgData) => {
      const imgURL = imgData.url;
      const img = document.createElement("img");
      img.src = imgURL;
      document.getElementById("image-container").appendChild(img);
    });
  })
  .catch((error) => console.error("Error fetching images:", error));

Code Examples

Below are code examples for working with images fetched through the API.

Fetch Image from Server and Display in UI


let isLoading = false;
let imgIndexIncrement = 0;

function fetchImageFromServer() {
  const endpoint = "/api/images-random";
  
  fetch(endpoint)
    .then((res) => res.blob())
    .then((blob) => {
      const imgURL = URL.createObjectURL(blob);
      document.getElementById("image").src = imgURL;
      isLoading = false;
    })
    .catch((error) => console.log(error));
}

// Trigger image fetch when page loads
document.addEventListener("DOMContentLoaded", fetchImageFromServer);

Styling

Here are some tips and examples for styling the images and layout using CSS.

Styling the Image Container

You can style the image container to adjust the width, height, margins, and borders to fit the design. 


#image-container {
  display: flex;
  flex-wrap: wrap;
  gap: 15px;
  justify-content: center;
  padding: 20px;
}

#image-container img {
  max-width: 100%;
  height: auto;
  border-radius: 12px;
  box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1);
  transition: transform 0.3s ease;
}

#image-container img:hover {
  transform: scale(1.05);
}

Centering Images

To ensure images are centered within the page, you can use the following CSS rules. 


body {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

img {
  margin: 0 auto;
  display: block;
}

Responsive Image Layout

Use media queries to create a responsive layout that adapts to different screen sizes. 


@media (max-width: 768px) {
  #image-container {
    flex-direction: column;
    gap: 10px;
  }

  #image-container img {
    width: 100%;
  }
}
